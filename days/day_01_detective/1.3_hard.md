# Day 01 — Hard | The Escape Route

> "The suspect has one shot to get out of the city. Every road has a cost. Find the cheapest way out — before he does."

## Problem

You are given a network of `n` cities labeled `1` to `n` and a list of directed roads `times`, where `times[i] = [u, v, w]` means traveling from city `u` to city `v` costs `w` minutes.

A detective is stationed at city `src`. He needs to send an alert to **every city** in the network.

Return the **minimum time** until all cities have received the alert. If any city is unreachable, return `-1`.

> This is harder than it sounds: the answer isn't the shortest path to one city — it's the shortest path to **all of them** simultaneously.

---

## Examples

```
Input:  n = 4, src = 2
        times = [[2,1,1],[2,3,1],[3,4,1]]
Output: 2
```

```
Input:  n = 2, src = 1
        times = [[1,2,1]]
Output: 1
```

```
Input:  n = 2, src = 1
        times = [[2,1,1]]
Output: -1
```

---

## Constraints

| | |
|---|---|
| Cities | `1 <= n <= 100` |
| Roads | `1 <= times.length <= 6000` |
| Road cost | `1 <= w <= 100` |
| Source | `1 <= src <= n` |
