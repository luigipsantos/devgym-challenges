# Day 01 — Medium | The Scrambled Messages

> "The gang isn't stupid, they scramble every message before sending. But patterns always betray the guilty."

## Problem

Given an array of strings `strs`, group the anagrams together. You can return the answer in any order.

> Two strings are anagrams if one can be formed by rearranging the letters of the other.

---

## Examples

```
Input:  strs = ["eat","tea","tan","ate","nat","bat"]
Output: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

```
Input:  strs = [""]
Output: [[""]]
```

```
Input:  strs = ["a"]
Output: [["a"]]
```

---

## Constraints

| | |
|---|---|
| Array length | `1 <= strs.length <= 10⁴` |
| String length | `0 <= strs[i].length <= 100` |
| Characters | lowercase English letters only |
